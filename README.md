# DND Custom Dice Static Site

This is a static B2B website for a wholesale 5e-compatible fantasy RPG dice supplier.

## Pages

- `index.html` - homepage and main B2B positioning
- `wholesale-dice.html` - wholesale landing page
- `custom-rpg-dice.html` - custom and private label dice page
- `fantasy-rpg-dice-themes.html` - theme hub
- `dragon-dice-wholesale.html` - dragon theme SEO page
- `character-class-dice.html` - character class theme SEO page
- `elemental-dice-sets.html` - elemental theme SEO page
- `dungeon-relic-dice.html` - dungeon and relic theme SEO page
- `sample-kits.html` - sample kit conversion page
- `custom-packaging.html` - packaging page
- `quality-manufacturing.html` - QC and manufacturing trust page
- `request-a-quote.html` - static inquiry page
- `faq.html` - SEO FAQ page
- `blog/` - SEO guide articles
